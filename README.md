
# 🧾 Dynamic Invoice Generator

A responsive web-based **Dynamic Invoice Generator** built using **HTML, CSS, and JavaScript** that enables users to create professional invoices with automatic calculations. The application allows users to add or remove products, calculate GST, discounts, subtotal, and grand total in real time, and generate printable invoices or downloadable PDF files.


## 📂 Project Structure

Dynamic-Invoice-Generator/
│── index.html

│── style.css

│── script.js

│── dashboard.css

│── dashboard.js

└── README.md

## ✨ Features

* 🏢 Professional company header with custom branding
* 📝 Customer details (Name, Phone, Email, Address)
* 🔢 Automatic Invoice Number & Date
* ➕ Add unlimited product rows
* 🗑️ Delete products dynamically
* 🧮 Automatic subtotal calculation
* 💰 GST calculation
* 🎯 Discount calculation
* 📊 Grand Total calculation
* 📄 One-click Print Invoice
* 📥 Download Invoice as PDF
* 📱 Fully Responsive Design
* 🎨 Modern and User-Friendly Interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* jsPDF
* html2canvas

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Fill in invoice details.
5. Add products and generate the invoice.
6. Print or download the invoice as a PDF.
